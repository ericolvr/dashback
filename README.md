# Dashboard Backend #
Consumer to process status sended by fog cannon


## Commands ##
All commands are in Makefile
```shell
1) make help
```
## Install Dependencies  ##
```shell
1) make install 
```

## Run Containers  ##
```shell
1) make containers 
```

## Run Local  ##
```shell
1) make run 
```

### Software ##
- Golang
- RabbitMQ
- MongoDB
- Websockets
- SSE ( Server Side Events )
